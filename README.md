# Minecraft-Rich-Text

Un plugin IntelliJ pour visualiser et tester le rendu des textes colorés et stylisés au format Minecraft (§).

## Fonctionnalités
- Aperçu en temps réel des textes avec codes couleurs et styles Minecraft (ex : §4, §l, §n, etc.)
- Prise en charge des couleurs (§0-§f) et styles (§l gras, §n souligné, §o italique, §m barré, §r reset)
- Interface simple dans une ToolWindow dédiée

## Exemple d'utilisation
```
§4Texte rouge §l§ngras et souligné §r§aVert normal
```
Affichera :
- "Texte rouge" en rouge
- "gras et souligné" en rouge, gras et souligné
- "Vert normal" en vert clair, style normal

## Installation
1. Cloner le dépôt :
   ```
   git clone https://github.com/BCK-Inc/Minecraft-Rich-Text.git
   ```
2. Ouvrir le dossier dans IntelliJ IDEA
3. Lancer le plugin via la configuration "Run Plugin" (Gradle)
4. Utiliser la ToolWindow "Minecraft Rich Text" pour tester vos textes
